# 夫婦Discord運用ガイド

## サーバー構成

### 📋 カテゴリ & チャンネル設計

#### 💬 日常コミュニケーション
- `#general` - 一般的な雑談
- `#today` - 今日の予定・出来事の共有
- `#quick-notes` - 簡単なメモ・リンク共有

#### 📅 生活管理
- `#calendar` - スケジュール共有・調整
- `#tasks` - タスク管理・TODO
- `#reminders` - リマインダー通知

#### 💰 家計管理
- `#expenses` - 支出報告・家計簿
- `#budget` - 予算管理・相談

#### 🏠 家事・生活
- `#shopping` - 買い物リスト
- `#maintenance` - 家のメンテナンス・修理

#### 🎮 趣味・エンタメ
- `#games` - ゲーム関連
- `#movies-books` - 映画・本の共有
- `#travel` - 旅行計画

#### 🤖 自動化
- `#bot-commands` - Bot操作用
- `#notifications` - 各種通知

## 運用のメリット

1. **履歴管理**: 大事な話を後から検索できる
2. **非同期コミュニケーション**: 相手の都合を気にせず連絡
3. **ファイル共有**: 画像・PDF等の簡単共有
4. **通知管理**: チャンネル別に通知レベル設定可能
5. **Bot活用**: 面倒な作業を自動化

## ランニングコスト

- Discord: **完全無料**
- Bot: **無料** (Herokuの無料枠 or 自宅PC)
- API利用: **月数百円程度** (使用量による)

**総コスト: 月0-500円程度**
